# EC-API

> Es handelt sich um eine API für den EC-Nordbund (https://www.ec-nordbund.de) die von Sebastian Krüger und Tobias Krause entwickelt wird.

## Befehle
```sh
# Installieren der Packete:
yarn

# Linten des Codes
yarn lint
# oder
yarn lint --fix

# Builden des codes
yarn build

# Typechecking
yarn types

# Starten
yarn start
```
